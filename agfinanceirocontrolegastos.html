Política de Privacidade – AG Finanças: Controle Gastos

O aplicativo AG Finanças: Controle Gastos é uma ferramenta de apoio para organização financeira pessoal. Ao utilizar o aplicativo, você concorda com os termos abaixo.

1. Uso do Aplicativo e Responsabilidade

O aplicativo é destinado ao registro e acompanhamento de informações financeiras. O usuário é responsável pelos dados que inserir e pelas decisões que tomar com base nessas informações.

Os cálculos realizados pelo aplicativo são automáticos e têm caráter informativo. Embora sejam desenvolvidos para oferecer apoio ao usuário, podem ocorrer variações, imprecisões ou inconsistências nos resultados. Por isso, é recomendável conferir os valores antes de realizar pagamentos ou tomar decisões financeiras.

O desenvolvedor não se responsabiliza por prejuízos, perdas, custos, cobranças, juros ou qualquer consequência relacionada ao uso do aplicativo, incluindo diferenças, possíveis erros de cálculo ou inconsistências nos valores apresentados.

2. Privacidade e Dados

As informações inseridas no aplicativo são armazenadas apenas no dispositivo do usuário.

O desenvolvedor não possui acesso aos dados inseridos no aplicativo.

O aplicativo não exige cadastro nem solicita informações pessoais como nome, CPF ou dados bancários para funcionar.

O aplicativo pode utilizar serviços de terceiros que realizam coleta de dados de forma independente, conforme descrito nesta política.

3. Publicidade (Google AdMob)

Para manter o aplicativo gratuito, são exibidos anúncios por meio do serviço Google AdMob.

O Google pode coletar identificadores técnicos do dispositivo, como o ID de publicidade, e outras informações necessárias para exibição e personalização de anúncios.

Esses dados são coletados e utilizados pelo próprio Google, de acordo com suas políticas de privacidade. O desenvolvedor não possui acesso a esses dados nem participa do tratamento dessas informações.

4. Controle e Exclusão de Dados

O usuário tem controle sobre suas informações:

  • Armazenamento Local: Todos os dados inseridos são salvos apenas no dispositivo.
  • Exclusão: O aplicativo permite excluir os dados locais a qualquer momento por meio das opções internas. A desinstalação do aplicativo geralmente resulta na remoção dos dados armazenados no dispositivo.

5. Funcionalidades do Aplicativo e Armazenamento de Dados

O aplicativo pode oferecer funcionalidades como compartilhamento do app, redirecionamento para avaliação na Google Play e opções de backup e restauração de dados.

O compartilhamento utiliza recursos do próprio dispositivo, sem coleta de dados pessoais. Ao acessar a Google Play, o usuário estará sujeito às políticas da plataforma.

Os dados permanecem armazenados no dispositivo do usuário. As funções de backup e restauração são de responsabilidade do usuário, incluindo a escolha de onde salvar essas informações.

O desenvolvedor não se responsabiliza pela perda de dados e informações.

6. Exportação de Dados

O aplicativo pode permitir a exportação de informações para arquivos em formatos como PDF e Excel.

Esses arquivos são gerados com base nos dados inseridos pelo usuário e podem ser salvos no dispositivo ou no local escolhido pelo próprio usuário.

O gerenciamento, armazenamento e compartilhamento desses arquivos são de responsabilidade do usuário.

7. Atualizações da Política de Privacidade

Esta política pode ser atualizada a qualquer momento para refletir melhorias no aplicativo ou ajustes necessários.
