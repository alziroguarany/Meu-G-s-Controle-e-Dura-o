Política de Privacidade

O aplicativo AG Financeiro - Controle Gastos respeita a sua privacidade e está comprometido com a proteção dos seus dados.

1. Coleta de Informações

O aplicativo não coleta dados pessoais identificáveis como:

  • Nome
  • E-mail
  • Endereço
  • Localização precisa

Todos os dados ficam armazenados apenas no seu dispositivo.

2. Dados Inseridos pelo Usuário

As informações financeiras registradas:

  • Ficam salvas localmente
  • Não são enviadas para servidores
  • Não são compartilhadas

3. Uso de Anúncios

O app pode exibir anúncios via serviços como Google AdMob, que podem coletar:

  • ID de publicidade
  • Interações com anúncios

Esses dados são usados apenas para melhorar a relevância dos anúncios.

4. Compartilhamento de Dados

O aplicativo não compartilha dados pessoais, exceto quando necessário para funcionamento de anúncios.

5. Armazenamento e Segurança

Os dados ficam no dispositivo do usuário. Apesar das boas práticas, não é possível garantir segurança absoluta.

6. Exportação, Backup e Compartilhamento

O aplicativo permite exportação e backup utilizando o sistema do Android.

  • Backup gerado localmente
  • Compartilhamento feito manualmente pelo usuário
  • Sem envio automático de dados
  • Restauração somente com ação do usuário

Importante: Após o envio, a segurança depende do serviço escolhido (E-mail, Drive, WhatsApp, etc.).

7. Direitos do Usuário

  • Excluir todos os dados
  • Parar de usar o app
  • Restaurar backup
  • Resetar informações

8. Alterações nesta Política

Esta política pode ser atualizada periodicamente.

9. Contato

Em caso de dúvidas, entre em contato pela página do desenvolvedor na loja.
