Antigravity · Guia de Instalação

ClaudeCode.

Guia completo para instalar e configurar o Claude Code no Windows — do zero ao primeiro prompt.

01
Criar conta ou gerar API Key

Você tem duas formas de autenticar o Claude Code — escolha a que preferir:

✦
Conta Claude Pro / Max
Recomendado para uso pessoal
MAIS FÁCIL

Se você já tem ou quer criar uma conta no Claude.ai (plano Pro ou Max), pode usar diretamente — sem precisar de API Key. Basta fazer login pelo próprio terminal.

🟠 claude.ai/login — Entrar com conta existente → ⭐ claude.ai/upgrade — Criar conta Pro / Max →
ou
🔑
API Key (Anthropic Platform)
Para desenvolvedores e automações
AVANÇADO

Crie uma conta na plataforma da Anthropic, gere uma API Key e use créditos por consumo. Ideal para integrar em projetos e pipelines.

🔷 console.anthropic.com — Criar conta na plataforma → 🔑 platform.claude.com/settings/keys — Gerar API Key →
02
Instalar Node.js

Baixe e instale o Node.js LTS. Após a instalação, verifique se está funcionando:

🟢 nodejs.org — versão LTS recomendada →
# Verificar instalação node --version npm --version
03
Instalar Git WINDOWS

O Git é necessário para algumas funcionalidades do Claude Code.

🔷 git-scm.com/downloads/win →
04
Configurar PowerShell

Permita a execução de scripts e adicione o npm ao PATH. Abra o PowerShell e rode:

Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
# Adicionar npm ao PATH (substitua SEU_USUARIO pelo seu usuário) $env:PATH += ";C:\Users\SEU_USUARIO\AppData\Roaming\npm"
05
Instalar Claude Code

Com o Node.js instalado, execute o comando abaixo para instalar o Claude Code globalmente:

npm install -g @anthropic-ai/claude-code
06
Iniciar e Autenticar

Execute o Claude Code no terminal. Ao iniciar pela primeira vez, ele vai perguntar como você quer autenticar:

claude
// DUAS OPÇÕES DE LOGIN O Claude Code aceita autenticação via conta Claude.ai (Pro/Max) ou via API Key. Escolha a opção que você configurou no Passo 01.
/login
Entrar com conta Pro ou API Key
/logout
Encerrar sessão atual
07
⚠️
Node.js falhou ao instalar ferramentas automáticas?
// ATENÇÃO Siga estes passos somente se o Node.js não instalou as ferramentas de build automaticamente durante sua instalação.
Passo A — Visual Studio Build Tools

Abra o PowerShell como Administrador e execute:

winget install Microsoft.VisualStudio.2022.BuildTools --override "--quiet --wait --norestart --add Microsoft.VisualStudio.Workload.VCTools --includeRecommended"

Se o winget falhar, instale manualmente pelo link abaixo:

🛠️ Download manual — vs_BuildTools.exe →
Passo B — Instalar Chocolatey

Abra o PowerShell como Administrador e rode os comandos na ordem:

# 1. Liberar execução de scripts Set-ExecutionPolicy Bypass -Scope Process -Force # 2. Ativar protocolo de segurança TLS [System.Net.ServicePointManager]::SecurityProtocol = [System.Net.ServicePointManager]::SecurityProtocol -bor 3072 # 3. Instalar Chocolatey irm https://community.chocolatey.org/install.ps1 | iex

Verifique se instalou corretamente:

choco -v
Passo C — Instalar Python

Baixe e instale o Python manualmente pelo link abaixo. Marque a opção "Add Python to PATH" durante a instalação.

🐍 Python 3.14.3 — Download direto (64-bit) →
